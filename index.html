Sintaxis

Literales

  • expresiones: 3+5
  • arreglos: [23, 59, 6, 8, 3]
  • objetos: {codigo:002, nombre:'Carlos', color:"verde"}
  • funciones: function myFunction() { document.write("Hola"); }

Variables


      var m;
      x = 10;
  • Se pueden declarar sin incializar con var
  • O se pueden incializar y quedan declaradas incluso sin var
  • No pueden empezar por un número
  • Son CASE-SENSITIVE

Objetos

  • Propiedades: nombreObjeto.propiedad o nombreObjeto["propiedad"]
  • Métodos: nombreObjeto.nombreMetodo()
  • Tres formas de crear un objeto:
    1. Lista de pares propiedad/valor: 
      
            var producto = {codigo:1, nombre:'botella', precio:10};
    2. Palabra reservada NEW: 
      
            var producto = new Object();
            producto.codigo = 1;
            producto.nombre = 'botella';
            producto.precio = 10;
    3. Función constructor: 
      
            function producto(codigo, nombre, precio) {
              this.codigo = codigo;
              this.nombre = nombre;
              this.precio = precio;
            }
            var producto1 = new producto(1, 'botella', 10);
  • Ejemplos de otros objetos predefinidos: 
    
          var n1 = new String();
          var b1 = new Boolean();
          var a1 = new Array();
          var r1 = new RegExp();
          var f1 = new Function();
          var d1 = new Date();

Funciones

  • Puede devolver un valor con la sentencia return
  • Ejemplo de función que se llama a sí misma (sin nombre de función): 
    
          (function () {
            alert('Esta función se invoca a sí misma');
          })();
  • Ejemplo de función asignada a una variable (sin nombre de función): 
    
          var z = function (x, y) {return x-y};
          var w = z(15, 6);

Operadores

  • Comparación igual al valor y al tipo ===
  • Comparación diferente al valor y al tipo !==
  • Aritmético módulo %
  • Aritmético incremento a = ++b
  • Aritmético decremento a =--b
  • Asignación incremento a +=b es lo mismo que a = a + b
  • Lógicos AND, OR y NOT && || !
  • Concatenación de cadenas + +=
  • Propiedades y métodos del objeto Math https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Objetos_globales/Math 
    
          var a = Math.PI; // a=3.14159
          var c = Math.max(1, 20); // c=20
          var d = Math.pow(2, 4); // d=16

Objeto DATE

  • Declaración
    
        var fecha1 = new Date(); // fecha y hora del sistema
        var fecha2 = new Date("December 26, 2014 12:30:20");
        var fecha3 = new Date(2015, 0, 16); // 16-ENE-2015 porque el mes es base cero
  • Propiedades
    
        fecha1.setFullYear(2020);
        fecha1.getFullYear(); // 2020
        fecha2.setMonth(11); // diciembre
        fecha2.setDay(30);
        fecha2.setHours(9);
